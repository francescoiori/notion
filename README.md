# notion

> Embeddable widgets for Notion pages.

## Widgets

### Timer

A minimal countdown timer. Adjust the duration with the arrow buttons, then click the circle to start. Click again to reset. Automatically matches your system's light/dark theme.

## Setup & Usage

1. Enable **GitHub Pages** on your repo (Settings → Pages → Source: `main`, folder: `/ (root)`)
2. Your widgets will be available at `https://<username>.github.io/notion/<widget>/`
3. In Notion, paste the URL and choose **Create embed**
4. Resize the widget to fit your page

> **Tip:** Set Notion's appearance to "Use system setting" so the widget theme matches automatically.

## Local Preview

Open any widget's `index.html` directly in your browser — no build step needed.
